# TweetFrost

TweetFrost is a custom, lightweight third-party client for Twitter (X), built from scratch for Android devices.

## Features
- Clean and modern interface.
- Optimized for arm64-v8a and armeabi-v7a architectures.
- Built for mobile-first experience on tablets.

## Goal
The goal of this project is to provide a smooth, minimalist way to interact with the platform while maintaining full control over the code.

## Development
- Language: Kotlin (or Dart, if you confirm)
- Architecture: MVVM (Model-View-ViewModel)
- Compatibility: Android 10+
